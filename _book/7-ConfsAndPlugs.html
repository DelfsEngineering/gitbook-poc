Configurations and Plugins

All configurations are stored as JSON in a file named book.json.

Relevant Fields:

  • Gitbook Version:
    { "gitbook": "2.x.x" }
  • Title :
    { "title": "My Awesome Book" }

If not specified it is extracted from README.md (First Line)

  • Description:
    { "description": "This is such a great book!" }

If not specified it is extracted from README.md (First Paragraph)

  • Styles:
    {
  "styles": {
      "website": "styles/website.css",
      "ebook": "styles/ebook.css",
      "pdf": "styles/pdf.css",
      "mobi": "styles/mobi.css",
      "epub": "styles/epub.css"
  }
}
  • Plugins :
    { "plugins": ["mathjax"] }

Plugins can be searched and downloaded from Plugins Site

  • Variables
    {
  "variables": {
      "myTest": "Hello World"
  }
}
  • Plugin Config :
    {
  "plugins": ["myplugin"],
  "pluginsConfig": {
      "myPlugin": {
          "message": "Hello World"
      }
  }
}

    This option contains all plugin specific details

  • Structure
    {
  "structure": {
      "readme": "INTRO.md"
  }
}
